包导入 (Package Imports)

使用 tsconfig paths

json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}

使用 package.json#imports

json
{
  "imports": {
    "#components/*": "./src/components/*.tsx",
    "#ui/*": "./src/components/ui/*.tsx",
    "#lib/*": "./src/lib/*.ts",
    "#hooks/*": "./src/hooks/*.ts"
  }
}

需要在 tsconfig.json 中启用 resolvePackageJsonImports 并使用 moduleResolution: "bundler"。
